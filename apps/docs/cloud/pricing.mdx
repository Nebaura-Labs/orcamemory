## FAQ

<AccordionGroup>
  <Accordion title="What counts as an API call?">
    Each memory read, write, search, or delete operation counts as one API call. Bulk operations count as multiple calls.
  </Accordion>
  <Accordion title="What happens if I exceed my limits?">
    On the Free plan, additional API calls will be rejected until the next day. Consider upgrading to Pro for unlimited calls.
  </Accordion>
  <Accordion title="Can I change plans anytime?">
    Yes, you can upgrade or downgrade at any time. Changes take effect immediately, and billing is prorated.
  </Accordion>
  <Accordion title="Is there an annual discount?">
    Yes! Pay annually and get 2 months free on Pro and Team plans.
  </Accordion>
  <Accordion title="Do you offer enterprise plans?">
    Yes, contact us at support@orcamemory.com for custom enterprise pricing with dedicated infrastructure, SLAs, and support.
  </Accordion>
</AccordionGroup>
